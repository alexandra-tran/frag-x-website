commands

This page lays out the specification for commands in frag-x

The basics

When sending a message there are two different types, the first is a normal message, this is the kind of message you want other players in the game to see. The other type of message is one that is a command, these are run to change something about the game.

When sending commands there are two different types of commands, there are client commands and server commands. To run a client command you must prefix your command like so: -c <command>, similarly for server commands you just do -s <command>

client commands

command what it does
/sens <NUMBER> changes your sensitivity to <NUMBER>

server commands

/map_vote <MAP NAME> starts a map vote, note: MAP NAME doesn't contain the extension
